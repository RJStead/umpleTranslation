Example of embedding an UmpleOnline page in an iFrame showing just a read-only diagram

The following umpleOnline iframe URL uses ?example=2DShapes&readOnly&notext&nomenu&nochrome

Height should to be specified, or else scrolling should be used. You can also show just text, a menu and various other combinations of UmpleOnline features. You can allow editing as part of a live document (although it is recommended setting up your own UmpleOnline server if you do that in your organization).
