@@description

<h2>Umple semantic error raised when a class does not contain the required attribute of a given type</h2>

<p>
Model constraints can be applied to Umple classes to ensure they respect certain properties, as defined in their constraints.
This error is raised if a class does not contain an attribute of a specified type in the constraint.
</p>
